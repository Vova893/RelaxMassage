<b><u>Название Telegram бота:</u></b> RelaxMassageBot  
<b><u>Краткое описание:</u></b> Telegram бот для записи на услуги массажиста  
<b><u>Публичная ссылка в Telegram:</u></b> @FeelGoodMassageBot

<b><u>Описание</u></b>  
При помощи этого бота пользователи могу т записаться на прием к массажисту выбрав дату и время удобное для них.
Посмотреть историю своих записей, и могут отменить запись. 
Также пользователи могут просмотреть прайс услуг.
Админ дополнительно может просматривать все записи пользователей, а также редактировать даты и время,
которые доступны для записей пользователя или не доступны.

<b><u>Старт</u></b>  
Версия Python: Python 3.13

<i>Создаем виртуальное окружение:</i>  
python -m venv venv  

<i>Активация виртуального окружения:</i>  
venv/Scripts/Activate.ps1  

<i>Устанавливаем все зависимости в Python из файла requirements.txt:</i>  
pip install -r requirements.txt

<u><b>Использование бота:</u></b>
/start - # Запускается бот и появляются три reply-кнопки у пользователя:  
'📅 Записаться на массаж' - здесь пользователь может выбрать дату и время подходящее для него, 
затем выбрать услугу и записаться на прием указав как его зовут и контактный номер телефона.  
'💰 Прайс услуг' - здесь пользователь может ознакомиться с прайсом и ценами услуг.  
'📋 История записи' - здесь пользователь может посмотреть историю своих записей и отменить запись если это нужно.  
у админа дополнительно появляются ещё две reply-кнопки:  
'🛠 Редактировать время' - здесь админ может редактировать даты и время, 
отмечать деты и определенное время которые не доступны для записи пользователя  
'📋 Посмотреть записи' - здесь админ может посмотреть все активные записи пользователей
